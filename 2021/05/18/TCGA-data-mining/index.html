TCGA data processing

Data download

  • 网页下载（最靠谱）
  • 命令行下载（可能会断，一定要校验）
  • R包下载（有问题，经过清洗）

下载文件：

  • Metadata（json文件，meta文件中含有样品号的TCGA barcode，而需要根据文件名转换成样本号）
  • Manifest, Cart（命令行下载需要这两个文件）

下载数据类型选择：

  1. htseq.counts是RNA-Seq原始reads文件，利用该数据通过不同的算法得到表达值（如果要DeSeq处理，下载这个）
  2. FPKM.txt.gz是存储每个样品所有测序基因的FPKM值（一般下这个）
  3. FPKM-UQ.txt.gz是存储每个样品所有测序基因的FPKM-UQ（upper quantile）值，即调整后FPKM值（用reads值3/4分位值替代原reads值）。

  • 行名：Ensembl ID

  • 第四部分：1-9为肿瘤，10以上为正常，判断正常及肿瘤的代码，常用的R函数 处理时需要注意，需要切割，取前面的整数

TME scoring

肿瘤微环境是肿瘤细胞所处的细胞环境，其组成包括细胞外基质、可溶性分子和肿瘤基质细胞。肿瘤微环境一旦形成，众多免疫细胞，如T细胞、髓源抑制性细胞，巨噬细胞等，都被趋化至此，构成肿瘤的微环境。在肿瘤微环境中，免疫细胞和基质细胞是两种主要类型的非肿瘤组分，并且已被提出对于肿瘤的诊断和预后评估是有价值的。

ESTIMATE (Estimation of STromal and Immune cells in MAlignant Tumor tissues using Expression data) is a tool for predicting tumor purity, and the presence of infiltrating stromal/immune cells in tumor tissues using gene expression data. ESTIMATE algorithm is based on single sample Gene Set Enrichment Analysis and generates three scores:

  • stromal score (that captures the presence of stroma in tumor tissue).
  • immune score (that represents the infiltration of immune cells in tumor tissue).
  • estimate score (that infers tumor purity).
