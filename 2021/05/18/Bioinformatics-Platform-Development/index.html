Class 1

超文本标记语言HTML

  1. Web发展史

  • 静态网页(Static Web Page)
    在网站设计中，纯粹HTML格式的网页通常被称为“静态网页”，文件扩展名是.htm、.html

  • 动态页面(Dynamic Page)
    页面代码没有变，但是显示的内容却是可以随着时间、环境或者数据库操作的结果而发生改变

  • AOP: Aspect Oriented Programming
    集成作法，先到block，判断权限，进入其中特定的页面

  • MVC: Model, View and Controller（e.g. PHP）

    View：视图，用户对view提出request（公司前台，用户与view接触，进行处理） Controller：接受信息，与数据库做交互，（产品经理） Model：处理信息（码农）

  • N Server + N Database - Browser
    负载均衡方案，不再是固定的访问某一个服务器
    把不同处理能力的服务器连在一起，用一个性能稍微好一点的引擎做管理

    • 冷备份：硬盘拷一下，传过去，硬盘传的时候，服务器要关掉，再上传到另外一个服务器
    • 热备份：实时传输，锁操作，但容易出问题
    • 温备份：既可以满足相应需求，又安全
  1. HTML实例
  • <html>元素定义了整个 HTML 文档
  • <head> 标签用于定义文档的头部，它是所有头部元素的容器。<head> 中的元素可以引用脚本、指示浏览器在哪里找到样式表、提供元信息等等。
    主要包括<base>,<link>, <meta>, <script>, <style>, 以及 <title> （定义文档的标题）
  • <body>元素定义了 HTML 文档的主体
    主要包括<p>, <table>, <img>, <a>, <h1>, <br>, <div>, <ul>和<canvas>, <code>, <video>等
  • <script>标签用于定义客户端脚本，比如 JavaScript（也可嵌入<body>中）
  • <style>标签用于为 HTML 文档定义样式信息，例如CSS等
